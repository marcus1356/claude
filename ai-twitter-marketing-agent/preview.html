AI Marketing Agent

Como rodar no seu computador

1. Clone: git clone https://github.com/marcus1356/claude.git
2. cd claude && git checkout claude/ai-twitter-marketing-agent-cYE4m && cd ai-twitter-marketing-agent
3. python3 -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt
4. cp .env.example .env && cp accounts.json.example accounts.json
5. python main.py e acesse http://localhost:8000
